## よく使う設定

### PROJECT_NAME

ドキュメントのタイトルを自由に設定する。メインページの見出しにも使われる。

### INPUT

入力ファイル。""で区切ること。

### EXCLUDE

入力から除くフォルダを指定する。

### OUTPUT_DIRECTORY

出力フォルダ

### HAVE_DOT

graphvizの設定

### GENERATE_LATEX

texを生成する
